# Midnight Signal v15.2.2 - Global + Watchlist Signals Hotfix

This hotfix restores the intended dashboard hierarchy:

- Global Top Signal is the primary market-wide signal.
- Top Signal in Your Watchlist is displayed alongside it.
- Watchlist personalization no longer replaces or hides the global leader.
- NIGHT remains the Midnight asset ticker.
- v15.2 live refresh behavior remains intact.

Run:

```bash
npm install
npm run build
```
