# v15.5 Conversion Intelligence

This release makes Global Top Signal actionable without overwhelming the dashboard.

## Key UX choices

- Removed the old standalone Top Signal hero.
- Promoted the two-panel system to the top of the app:
  - Your Top Signal = personal control and depth.
  - Global Top Signal = discovery, comparison, and conversion.
- Added Add to Watchlist and Track Signal flows so a global opportunity can become part of the user's own system.
- Added a gated Pro CTA around deeper global analytics instead of duplicating every panel.

## Why this matters

v15.5 closes the gap between discovery and ownership. Users can now see when the broader market has a better signal than their watchlist, understand why, and act on it.
