# Midnight Signal v15.4 - Dual Signal Discovery

This build separates personal relevance from objective discovery so users keep control of their watchlist without missing the stronger global opportunity.

## What changed

- Renamed the primary card to **Your Top Signal** so users understand it is selected from their watchlist.
- Added a separate **Global Top Signal** discovery hero for the best signal across the full asset universe.
- Added smart exposure logic that only emphasizes the global signal when it is materially stronger, high-confidence, or the user is Pro.
- Added Pro upgrade CTA around global analytics instead of replacing the user's own watchlist flow.
- Updated feedback copy so the learning loop clearly covers both personal and global signal context.

## Product principle

The app now makes two truths visible:

1. **Your Top Signal**: the best signal from assets the user already chose.
2. **Global Top Signal**: the strongest opportunity outside or beyond the user's watchlist.

This avoids distracting the user while still giving Midnight Signal a clean discovery and monetization path.
