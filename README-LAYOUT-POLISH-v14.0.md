# Midnight Signal v14.0 — Layout Polish + Product Clarity

This build focuses on product clarity rather than new feature sprawl.

## What changed

- Added a dominant “Tonight’s signal” hero above the rest of the dashboard.
- Moved proof/performance language higher so users see trust signals earlier.
- Tightened header copy to reduce explanation fatigue.
- Renamed the old brief area into a performance proof block.
- Simplified the public grid language from “Top 20 Signal Grid” to “Top Signal Grid.”
- Reduced the Pro CTA copy into one clearer upgrade path.
- Updated build metadata to v14.0.0.

## Deploy

```bash
npm install
npm run build
vercel --prod
```
